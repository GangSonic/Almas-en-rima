# Product Overview

**Altar de Palabras** is an interactive web application celebrating the Mexican Day of the Dead (Día de Muertos) tradition through personalized literary calaveritas and a fully interactive virtual altar.

## Core Features

- **Interactive Drag & Drop Altar**: 3-level traditional altar structure with grid-based positioning system
- **AI-Generated Calaveritas**: Personalized literary verses using Google Gemini AI with multiple tones (humorous, respectful, satirical)
- **Photo Upload System**: Upload and integrate photos of loved ones into the altar
- **Element Management**: 20+ draggable elements across 6 categories (candles, flowers, food, fruits, photos, decorations)
- **Persistence**: Auto-save functionality with localStorage and export/import capabilities
- **History System**: Tracks last 3 generated calaveritas with restore functionality

## Cultural Context

The application honors Mexican Day of the Dead traditions by:
- Creating personalized literary calaveritas (humorous death verses)
- Building virtual altars (ofrendas) with traditional elements
- Using authentic color palette inspired by cempasúchil flowers and traditional decorations
- Maintaining respectful tone while celebrating life and memory

## Target Experience

Users can create meaningful digital altars to honor deceased loved ones while engaging with traditional Mexican literary and cultural practices in an interactive, modern format.
